### 4.3 對TAICERA的機會與風險

| 面向 | 分析 |
|------|------|
| ✅ **機會** | 泰國市場對中國磁磚的**信任度正在下降**，品質投訴增加，買方開始尋找「替代中國」的來源 |
| ✅ **機會** | 泰國加強品質稽核，對不合TIS標準的低價中國磚設阻，反而利好合規的越南製品 |
| ✅ **機會** | 泰國本地廠減產→供給缺口擴大→進口需求持續成長（CAGR 18%） |
| ⚠️ **風險** | 若泰國實施廣泛的防衛關稅，可能連帶影響越南進口品 |
| ⚠️ **風險** | 中國在Temu等電商平台的D2C（直達消費者）模式，壓縮傳統經銷利潤 |

### 6.2 定價原則

| 原則 | 說明 |
|------|------|
| **不打價格戰** | 絕不與中國低端磚在 $3/m² 以下競爭，會犧牲品牌形象 |
| **錨定本地中端** | 以Campana（SCG子品牌）零售價為錨點，定價低10–15% |
| **分層報價** | 對代理商25%折扣、工程採購額外3%、新客戶免費樣品 |
| **以泰銖計價** | 最終報價可考慮用THB報價，消除泰方匯率顧慮 |
| **強調總成本優勢** | FOB+運費+0%關稅（ATIGA）= 泰國批發價，比中國含反傾銷風險的總成本更有保障 |

| 成本/環節 | 金額（USD/m²） | 比例 |

| TAICERA FOB出廠價 | $5.50 | — |
| 海運（胡志明→林查班） | $0.15 | — |
| 保險 | $0.05 | — |

| 泰國進口關稅（ATIGA） | **$0.00** | 0% |
| 泰國VAT（7%） | $0.40 | 7% |
| **泰國含稅到倉成本** | **$6.10** | — |

> 此價格落在「入門至中低端」區間，**低於Campana（฿350+）15–18%**，具有極強競爭力，同時避開中國最低價帶。
