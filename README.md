## 📋 專案摘要

本專案為越南同奈省磁磚製造商 **TAICERA** 開拓泰國出口市場所進行的系統性商業情報研究，涵蓋市場規模、法規合規、客戶開發策略、目標客群分析、歷年數據統計、潛在客戶名錄，以及價格與競爭分析等七大面向。

### 背景

- 🏭 **TAICERA** 位於越南同奈省（Đồng Nai），為越南磁磚製造商
- 🌏 全球經濟放緩、貿易關稅壁壘升高、原物料價格上漲
- 🇹🇭 目標積極開拓泰國磁磚出口市場
- 📊 泰國磁磚市場規模約 **USD 14.2 億**（2025年），進口年均成長率 **18%**（CAGR 2020–2024）

## 🔑 核心發現

### 市場機會

| 指標 | 數值 | 意義 |
|------|------|------|
| 泰國磁磚市場規模（2025） | **USD 14.2 億** | 東南亞重要市場 |
| 磁磚進口年均成長率 | **+18.06%**（CAGR 2020–2024） | 進口需求持續擴大 |
| 越南磁磚佔泰國進口量 | **14.3%**（第三大來源國） | 已有市場基礎 |
| ATIGA 優惠關稅稅率 | **0%** | 憑 Form D 完全免稅 |
| 越南→泰國海運天數 | **3–5 天** | 地理優勢明顯 |

### 競爭態勢

| 指標 | 數值 | 意義 |
|------|------|------|
| 中國磁磚佔泰國進口量 | **66.7%** | 最大競爭者但品牌力弱 |
| 中國磁磚低於泰國出廠價 | **30%** | 低價傾銷引發產業震盪 |
| SCG Ceramics 本地市佔率 | **~58%** | 本地龍頭，但正向高端轉移 |
| 越南磁磚 CIF 均價 | **$4.25/m²**（高於中國 $3.91） | 越南定位中端、品質溢價 |

### 策略建議

```
定位：「品質優於中國、價格低於泰國本地品牌」的中端性價比路線
渠道：優先尋找泰國在地進口代理商 → 參展 Architect Expo → 爭取零售通路代工
時程：18 個月內建立穩定出口渠道，目標年出口額 USD 100 萬
```
