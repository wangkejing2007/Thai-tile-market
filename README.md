## 📋 專案簡介

本專案為越南磁磚製造商 **TAICERA** 系統性開拓泰國市場所進行的完整商業情報研究，涵蓋六大面向：

> **市場規模 → 法規合規 → 業務策略 → 客群分析 → 客戶名錄 → 價格與競爭**

泰國磁磚市場規模約 **USD 14.2 億**（2025年），進口量CAGR達 **18%**（2020–2024），越南已是第三大進口來源國（佔14%）。TAICERA可藉由ASEAN自貿協定（ATIGA）享有 **0%進口關稅**，加上3–5天短海運距離，在泰國中端磁磚市場建立具競爭力的出口渠道。
