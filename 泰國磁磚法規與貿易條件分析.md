# 泰國磁磚出口法規與貿易條件深度分析
**報告對象：** TAICERA（越南同奈省）  
**報告日期：** 2026年2月22日  
**用途：** 開拓泰國磁磚出口市場之法規遵循指南

---

## 一、泰國磁磚進口法規框架概述

泰國對磁磚進口的管理主要涉及以下三個層面：

| 層面 | 主管機關 | 核心要求 |
|------|---------|---------| 
| **品質標準** | 泰國工業標準局（TISI） | 強制性TIS 2508-2555認證 |
| **關稅與貿易協定** | 泰國海關局（Thai Customs Department） | ATIGA優惠稅率、Form D |
| **進口申報** | 泰國海關局 | 進口報關文件、增值稅（7%）繳納 |

## 二、泰國強制品質標準：TIS 2508-2555

### 2.1 標準概述

泰國工業標準局（**TISI**，Thailand Industrial Standards Institute）要求所有進口磁磚必須符合 **TIS 2508-2555** 強制認證，此標準於 **2014年1月15日** 正式生效，整合並取代了原有的陶瓷地磚、牆磚及外牆磚分散標準。

> ⚠️ **重要：TIS認證屬強制性（Mandatory），缺乏此認證的磁磚不得在泰國市場銷售，海關可拒絕放行。**

### 2.2 TIS 2508-2555 涵蓋範圍

| 適用範圍 | 不適用範圍 |
|---------|----------|
| 擠壓成型與壓製成型的釉面及無釉陶瓷地磚和牆磚 | 小於規定尺寸的磁磚 |
| 外牆磁磚 | 高吸水率特殊磁磚 |
| 馬賽克磁磚（part） | 天然石材磁磚 |

### 2.3 主要測試項目與規格要求

| 測試項目 | 說明 | 相關HS品類 |
|---------|------|----------|
| **尺寸與表面平整度** | 長、寬、厚度公差，邊直度、直角度、平整度 | 全品類 |
| **吸水率（Water Absorption）** | 分為≤0.5%（磁化磚）、0.5–10%、＞10% | 全品類 |
| **斷裂模數與破壞強度** | 評估結構強度 | 全品類 |
| **深度耐磨性** | 無釉磁磚專用測試 | 無釉磁磚 |
| **表面耐磨性（PEI等級）** | 釉面磁磚防磨等級 | 釉面磁磚 |
| **耐龜裂性（Crazing）** | 釉面是否出現細裂紋 | 釉面磁磚 |
| **化學耐性** | 耐酸鹼化學品腐蝕 | 全品類 |
| **耐污漬性** | 防污易潔性能 | 全品類 |
| **表面外觀（1級品質）** | 無裂縫、龜裂、針孔、污漬或釉面缺陷 | 全品類 |

### 2.4 TIS認證申辦流程（進口商角度）

```
第1步：線上申請（TISI e-license系統）
   → 申請進口樣品授權、提交公司文件、發票副本、產品規格書

第2步：樣品進口測試
   → 透過泰國國家單一窗口（NSW）辦理樣品進口
   → 樣品送交TISI進行TIS 2508-2555標準測試

第3步：工廠品質管制系統評估
   → TISI派員或認可驗廠機構赴TAICERA同奈省廠區進行品質管制評估
   → 若TAICERA已在TISI登記，可提供「合規聲明書（LoC）」替代

第4步：正式申請TISI許可證（Mor.Or.5）
   → 提交產品測試報告 + 品質管制評估報告/LoC
   → 取得TISI許可證，許可證效期通常為3年

第5步：產品標示合規
   → 每批產品包裝須標示：TISI標誌、標準編號、製造商名稱/商標、
     產品名稱、型號、規格、生產日期
```

> 💡 **TAICERA行動建議：** 應立即委託泰國認可的在地驗廠公司（如 **SGS Thailand、Bureau Veritas Thailand、TÜV Rheinland Thailand**）協助辦理TIS認證。整體申辦週期約 **3–6個月**，建議優先啟動。

### 2.5 TIS認證常見問題

| 問題 | 解決方式 |
|------|---------|
| 樣品測試不合格 | 可調整配方或生產製程後重新送測 |
| 工廠評估不通過 | 強化品質管制文件、SOP、ISO 9001認證有助加分 |
| 認證費用 | 約THB 50,000–200,000（依測試項目數量而定） |

## 三、ATIGA Form D 申辦詳細流程

### 3.1 Form D（原產地證明書C/O）概述

**ATIGA（東協貨品貿易協定）Form D** 是越南出口商申請東協優惠關稅的核心文件。憑此文件，越南磁磚出口至泰國可享 **0%進口關稅**（否則適用MFN稅率10–20%）。

> ✅ **2024年關鍵更新：** 自2024年1月1日起，所有東協成員國已全面實施電子Form D（e-Form D），透過**東協單一窗口（ASW，ASEAN Single Window）**自動傳輸至進口國海關系統，不再依賴紙本。

### 3.2 ATIGA原產地規則（TAICERA適用性評估）

越南製磁磚需符合以下**任一**原產地條件才可申請Form D：

| 原產地規則 | 說明 | TAICERA適用性 |
|------------|-----|--------------|
| **完全獲得（WO）** | 產品完全在越南境內生產 | ✅ 極度適用（磁磚在同奈省製造） |
| **區域價值含量（RVC）≥ 40%** | 越南境內附加價值佔FOB價格40%以上 | ✅ 本地製造通常可輕鬆達標 |
| **關稅稅則改變（CTC）** | 原料與成品HS Code在4碼層次發生改變 | ✅ 通常適用（原料→成品加工） |

**結論：** TAICERA在同奈省自行生產磁磚，屬於「完全獲得」類別，Form D申請資格無疑問。

### 3.3 Form D申辦步驟（越南出口商操作指南）

```
【準備文件階段】
 ✔ 商業發票（Commercial Invoice）
 ✔ 裝箱單（Packing List）
 ✔ 提單/運輸文件副本（B/L or AWB）
 ✔ 已清關的越南出口報關單
 ✔ 生產製程說明書
 ✔ 原料採購發票（國內採購或進口報關單）
 ✔ 區域價值含量計算表（若採RVC方式）

【線上申請階段】
 → 登入越南工業貿易部 eCoSys 系統
   網址：ecosys.gov.vn
 → 選擇「Form D（ATIGA）」填寫申請表
 → 上傳所有準備文件

【審核與核發】
 → 工業貿易部核發局（越南各地出口管理辦公室）審核
 → 通常 3 個工作天內完成核發
 → e-Form D 自動透過東協單一窗口（ASW）傳輸至泰國海關
 → 無需再提交紙本（除非系統技術問題）

【注意事項】
 ⚠ 貨物價值 USD 200以下可豁免Form D要求
 ⚠ 若Form D在出口後補發，需在「Box 13」勾選「Issued Retroactively」
 ⚠ e-Form D資料一旦傳輸，泰國海關自動核對，不匹配將被扣押
```

### 3.4 Form D費用與時效

| 項目 | 說明 |
|------|------|
| 申辦費用 | 約VND 20,000–50,000/份（約1–2美元，費用極低） |
| 核發時效 | 通常3個工作日以內 |
| 有效期限 | 自核發日起 **12個月** 內有效 |
| 適用次數 | 一份Form D原則上對應一批貨物 |

## 四、越南出口報關流程

### 4.1 越南出口磁磚完整流程

```
Step 1：簽訂出口合約（Sales Contract）
   → 明確貿易條件（FOB/CIF）、交貨期、付款方式

Step 2：準備出口文件
   → 商業發票、裝箱單、提單、原產地證明（Form D）

Step 3：電子出口報關申報（E-Declaration）
   → 使用越南海關電子申報系統（VNACCS/VCIS）
   → 申報表格：HQ/2015/XK（出口申報表）
   → 申報內容：HS Code、數量、重量、FOB價值、目的地等

Step 4：海關分類（通關道）
   → 🟢 綠色通道：直接放行，無需查驗
   → 🟡 黃色通道：審查文件，不開箱
   → 🔴 紅色通道：開箱實物查驗

Step 5：繳納出口稅（磁磚通常免出口稅）
   → 越南對磁磚出口一般不課出口稅

Step 6：貨物裝船/交運
   → 胡志明市附近主要港口：
   • 胡志明港（Cảng Sài Gòn）
   • 蓋梅港（Cảng Cái Mép）—適合大型集裝箱船
   • 隆安港（Cảng Long An）—同奈省附近

Step 7：取得提單（Bill of Lading）
Step 8：向銀行結匯（出口退稅申請，如適用）
```

### 4.2 越南磁磚出口需備文件清單

| 文件名稱 | 必要性 | 發出單位 |
|---------|--------|---------|
| 電子出口報關單（E-Form HQ/2015/XK） | ✅ 必備 | 出口商自行申報 |
| 商業發票（Commercial Invoice） | ✅ 必備 | TAICERA開立 |
| 裝箱單（Packing List） | ✅ 必備 | TAICERA開立 |
| 提單（Bill of Lading） | ✅ 必備 | 船公司開立 |
| 銷售合約（Sales Contract） | ✅ 必備 | 買賣雙方簽署 |
| Form D（ATIGA原產地證明） | ✅ 必備（享0%關稅） | 越南工業貿易部核發 |
| 品質合格證書（COQ） | 🔷 建議取得 | 越南認可檢測機構 |
| TIS認證文件副本 | 🔷 建議附上 | TISI（泰國）核發 |
| 公司營業執照副本 | ✅ 首次出口必備 | 越南工商局 |

## 五、泰國海關申報與清關程序

### 5.1 泰國進口清關流程

```
Step 1：進口商於貨到前向泰國海關提交進口申報
   → 使用泰國電子通關系統（e-import / NSW）
   → 申報HS Code、貨值（CIF）、數量

Step 2：提交優惠關稅申請
   → 附上電子Form D（透過ASW自動傳輸）
   → 申請ATIGA 0%優惠稅率

Step 3：繳納進口增值稅
   → 增值稅（VAT）= CIF貨值 × 7%
   → 即使關稅為0%，VAT仍須繳納

Step 4：海關查驗分類
   → 依電腦風險評估系統分配通關道

Step 5：貨物放行
   → 完成清關後，泰國進口商可提取貨物

Step 6：進口商確認符合TIS標準
   → 首次進口須確認TIS認證文件完整
```

### 5.2 進口稅費彙總（越南→泰國）

| 稅費項目 | 稅率 | 計算基礎 | 說明 |
|---------|------|---------|------|
| 進口關稅（ATIGA） | **0%** | CIF價值 | 需持Form D |
| 進口關稅（MFN） | 10–20% | CIF價值 | 無Form D時適用 |
| 增值稅（VAT） | **7%** | CIF+關稅 | 所有進口必繳 |
| 其他費用 | 視情況 | — | 倉儲、代理費等 |

> **試算範例：** 一批CIF價值 USD 100,000 的磁磚
> - 持Form D：關稅 = $0，VAT = $7,000，**合計稅費 $7,000**
> - 無Form D（MFN 15%）：關稅 = $15,000，VAT = $8,050，**合計稅費 $23,050**
> - **Form D省下 $16,050！**（佔總貨值16%）

## 六、反傾銷與貿易救濟措施現況

### 6.1 泰國對磁磚的反傾銷歷史

| 時間 | 對象 | 內容 |
|------|------|------|
| 1990年代–2016年 | **中國磁磚** | 泰國曾對中國牆面及地磚課徵反傾銷稅，並於2016年延長 |
| 2016年後 | 中國磁磚 | 持續監控，部分品類仍受貿易救濟措施保護 |
| **2024年至今** | **越南磁磚** | ✅ **目前無反傾銷調查**，越南出口泰國處於安全窗口 |

### 6.2 越南磁磚的貿易風險評估

| 風險類型 | 風險等級 | 說明 |
|---------|---------|------|
| 泰國對越南磁磚反傾銷調查 | 🟢 **低風險** | 目前無調查記錄，越南僅佔14%進口量 |
| 中國磁磚洗產地（貿易轉移）疑慮 | 🟡 **中風險** | 需確保TAICERA產品100%越南本地製造，Form D原產地合規 |
| 未來泰國可能啟動調查 | 🟡 **中長期需關注** | 若越南出口量大幅增加，泰國本地廠商可能申請調查 |

### 6.3 如何降低反傾銷風險

- ✅ **保持Form D合規**：確保原材料進口及生產加工記錄完整，可供查核
- ✅ **推動公平定價**：出口FOB價格不應低於越南國內銷售價格（避免傾銷定義）
- ✅ **關注泰國貿易政策動態**：定期追蹤泰國商務部公告（www.dbd.go.th）
- ✅ **尋求法律諮詢**：如市場份額快速增長，建議委託貿易法律顧問進行風險評估

## 七、合規行動計畫（TAICERA建議時程）

| 時程 | 行動項目 | 負責單位 | 優先級 |
|------|---------|---------|--------|
| **立即（0–1個月）** | 委託泰國驗廠機構評估TIS認證需求 | 業務部+品管部 | 🔴 最高 |
| **立即（0–1個月）** | 向越南工業貿易部登記Form D申辦資格 | 貿易部門 | 🔴 最高 |
| **短期（1–3個月）** | 準備TIS測試樣品，送交TISI測試 | 品管部 | 🔴 最高 |
| **短期（1–3個月）** | 完成第一批出口文件範本（Invoice, PL, B/L） | 貿易部門 | 🟠 高 |
| **中期（3–6個月）** | TIS認證取得 | 品管部 | 🔴 最高 |
| **中期（3–6個月）** | 尋找泰國在地進口代理商 | 業務部 | 🟠 高 |
| **中期（6–12個月）** | 建立穩定的Form D申辦SOP | 貿易部門 | 🟡 中 |
| **長期（12個月+）** | 監控泰國反傾銷政策，必要時聘用貿易律師 | 管理層 | 🟡 中 |

## 九、總結

TAICERA出口泰國的法規遵循核心在於兩項：

| 優先順序 | 任務 | 效益 |
|---------|------|------|
| 🥇 **第一優先** | 取得泰國 **TIS 2508-2555 認證** | 合法進入泰國市場的入場券 |
| 🥈 **第二優先** | 建立 **ATIGA Form D** 申辦標準流程 | 每批貨物節省進口關稅10–20%，競爭力大幅提升 |

只要完成這兩項合規要求，TAICERA即具備以**完整合法身份、零關稅優勢**進入泰國磁磚市場的條件。
