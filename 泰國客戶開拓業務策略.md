## 三、渠道策略（How to Reach Them）

### 3.1 渠道一：尋找泰國在地代理/進口商 ⭐最優先

**這是開拓泰國市場最快速、成本最低的路徑。**

#### 目標代理商畫像
- 已具備磁磚或建材進口許可
- 有穩定的B2B客戶群（承包商、建商）
- 對越南製品接受度高
- 具備倉儲與物流能力

#### 已知潛在泰國進口商（可初步接洽）

| 公司名稱 | 類型 | 備註 |
|---------|------|------|
| **Herial International Co., Ltd.** | 建材進口商 | 泰國已知磁磚進口商 |
| **Bangkok Material Co., Ltd.** | 建材批發商 | 長期活躍進口買家 |
| **Trio Marketing Management & Services** | 貿易代理 | 已知對越南產品有採購記錄 |
| **Ruampat Ceramic Co., Ltd.** | 磁磚進口/經銷商 | 專注磁磚與裝飾建材 |
| **Antovia Surfaces / Lexira Surfaces** | 磁磚經銷商 | 代理商型態，尋找進口來源 |

#### 代理商開發行動步驟
```
Step 1：準備代理商開發資料包
  → 英文/泰文產品目錄（含尺寸、規格、MOQ）
  → FOB價格表（試探性報價，留有議價空間）
  → TAICERA公司介紹（年產能、ISO認證、主要客戶）
  → 樣品寄送計畫

Step 2：多渠道接觸潛在代理商
  → LinkedIn 直接訊息
  → 電子郵件 Cold Email（英文）
  → 透過越南商工總會（VCCI）介紹
  → 利用 Volza / Trademo 貿易數據平台找到具體採購決策人

Step 3：邀請來訪或安排視訊會議
  → 提供同奈省工廠參觀邀請（可搭配越南考察行程）
  → 視訊展示生產線、品質控制流程

Step 4：簽訂代理協議
  → 明確：獨家/非獨家地區代理
  → 約定：最低年採購量（MOQ）、付款條件
  → 提供：樣品支持、技術文件、行銷素材
```

## 四、產品策略（What to Sell）

### 4.1 建議主力出口產品線

| 產品類型 | 規格建議 | FOB建議價格 | 目標客群 | 理由 |
|---------|---------|-----------|---------|------|
| **瓷質地磚（Porcelain Floor Tile）** | 60×60cm、80×80cm | USD 5–9/m² | 建商、承包商 | 泰國市占最高（41%），需求量大 |
| **大板磚（Large Format Tile）** | 60×120cm、80×160cm | USD 8–14/m² | 飯店、豪宅開發商 | 高端趨勢，附加價值高 |
| **戶外防滑磚（Anti-slip Outdoor）** | 30×60cm、40×60cm | USD 4–7/m² | 工業廠房、商辦 | EEC工業區需求旺盛 |
| **木紋磚（Wood-look Tile）** | 20×120cm、15×90cm | USD 6–11/m² | 室內設計師、住宅 | 設計感強，差異化明顯 |
| **素色牆磚（Plain Wall Tile）** | 30×60cm、25×75cm | USD 3–6/m² | 住宅、飯店浴室 | 需求量大，適合批量出口 |

### 4.2 產品定位策略

```
                    高品質
                      ↑
                      │
         SCG Ceramics  │  ← TAICERA目標區間（性價比中高端）
         Dynasty Ceramic│
                      │
低價 ────────────────────────────────── 高價
                      │
         中國低端磁磚  │
                      │
                      ↓
                    低品質
```

**TAICERA定位：** 品質接近泰國中高端本地品牌，價格比本地品牌低20–30%，比中國高端品牌有越南製造的形象優勢（ATIGA、ESG、ASEANmade）。

## 五、定價策略（Pricing Strategy）

### 5.1 出口定價架構

| 成本項目 | 佔FOB價格比例 | 說明 |
|---------|-------------|------|
| 生產成本 | 55–65% | 原料、人工、能源 |
| 工廠毛利 | 15–20% | TAICERA利潤空間 |
| 出口費用 | 5–8% | 報關、裝運、包裝 |
| 銷售/行銷費用 | 3–5% | 展覽、樣品、佣金 |
| **FOB售價** | **100%** | 基準報價 |

### 5.2 代理商佣金結構建議

| 銷售模式 | 佣金/折扣率 | 說明 |
|---------|----------|------|
| 獨家區域代理商 | FOB價格給予20–25%折扣 | 換取最低採購量承諾 |
| 非獨家代理商 | FOB價格給予15–18%折扣 | 多家代理，自由競爭 |
| 直接工程採購（超過5,000m²） | 額外2–3%折扣 | 激勵大單 |
| 樣品 | 免費提供（含運至泰國） | 前3次採購方可享受 |

### 5.3 付款條件建議

| 客戶關係 | 建議付款條件 |
|---------|------------|
| **新客戶（前3單）** | 100% T/T提前付款，或信用狀（L/C） |
| **合作6個月以上** | 30% 訂金 + 70% 見提單副本付款 |
| **長期穩定客戶** | Net 30天（須信用調查後核准） |

## 十、風險管理

| 潛在風險 | 影響程度 | 應對策略 |
|---------|---------|---------|
| TIS認證取得延遲 | 🔴 高 | 提前6–8個月啟動，委託專業代辦機構 |
| 代理商銷售力道不足 | 🟠 中 | 設定KPI+合約中明訂最低採購量，達不到即可終止獨家 |
| 匯率波動（VND/THB） | 🟡 中 | 報價以USD計，避免匯率風險轉嫁 |
| 中國磁磚降價競爭 | 🔴 高 | 強化品質、設計、服務差異化，勿陷入價格戰 |
| 泰國建設市場放緩 | 🟡 中 | 分散客群（工業+住宅+飯店），非單一依賴一個市場 |
| 代理商倒帳風險 | 🟠 中 | 新客戶堅持預付款或L/C，6個月後再評估賒帳 |
